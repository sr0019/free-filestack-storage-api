# **free-filestack-storage-api**

This Cloudflare Worker script allows you to upload files of any size to filestack storage api and access them anytime via a unique URL. It provides a lightweight and efficient way to handle cloud storage.

## 🚀 Features
- **Unlimited File Size:** Upload files of any size seamlessly.
- **Permanent Access:** Files are accessible anytime via a generated URL.
- **Edge Powered:** Built on Cloudflare Workers for global low-latency access.

## 🛠️ How to Use

1. **Configuration:**
   - Open the `worker.js` file.
   - Locate the variable `your_worker_url`.
   - Update it with your actual worker URL, or keep it as it is (it will still function).
2. **Deployment:**
   - Log in to your [Cloudflare Dashboard](https://dash.cloudflare.com/).
   - Create a new Worker.
   - Paste the code from `worker.js` into the Cloudflare Worker editor.
   - Deploy the script.
3. **Start Using:** Once deployed, you can begin uploading and retrieving files.
