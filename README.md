# **free-filestack-storage-api**

This nodejs/expressjs script allows you to upload files of any size to filestack storage api and access them anytime via a unique URL. It provides a lightweight and efficient way to handle cloud storage.

## 🚀 Features
- **Unlimited File Size:** Upload files of any size seamlessly.
- **Permanent Access:** Files are accessible anytime via a generated URL.

## 🛠️ How to Use

### 1. Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) (v14 or higher) installed.
- Access to a terminal or command prompt.

### 2. Installation
Create a new folder, clone the repo.

```bash
# Initialize the project
npm init -y

# Install required dependencies
npm install express axios body-parser
### 3. Configuration
Open server.js.

Locate the YOUR_DOMAIN constant:

Local Testing: Keep it as http://localhost:3000.

Production: Change it to your live URL (e.g., https://your-app.com).

### 4. Deploy locally

node server.js
